Warning

There is a known issue with version 2.0.0 of the summarization software when configuring resources with PCP.

Be sure to add the following line to the resource configuration to ensure job-level CPU information is processed properly. ```sh "plugin_blacklist": ["ProcPrometheus"] ```

RPM installs

  • Default config location: /etc/supremm/config.json

Source code installs

  • Default config location: $INSTALL_PREFIX/lib/python3.6/site-packages/supremm-2.0.0-py3.6-linux-x86_64.egg/etc/supremm/config.json
